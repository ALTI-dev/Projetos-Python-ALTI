##  ALTI QUIZ v1.0

**Quiz interativo de conhecimentos gerais** com 3 níveis de dificuldade e sistema de classificação.

###  Jogar Online
[ **Jogar ALTI QUIZ** ](https://bit.ly/3LIJuW7)

>  **Nota:** Clica em "Run" no Replit para iniciar. Primeira execução pode demorar ~30 segundos (servidor gratuito).

###  Características
-  30 perguntas únicas
-  3 níveis de dificuldade (Fácil/Médio/Difícil)
-  Sistema de classificação com emojis
-  Feedback imediato para cada resposta
-  Modo contínuo ou jogo único

###  Executar Localmente
```bash
# Requisitos: Python 3.x
python quiz_de_conhecimento.py
```

###  Tecnologias
- Python 3.13
- Estruturas de dados (listas, tuplas)
- Controle de fluxo (loops, condicionais)
- Manipulação de strings
